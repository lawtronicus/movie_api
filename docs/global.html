(async) isUnique(field, value, userIdopt) → {Promise.<boolean>}

Checks if a specific field in the Users collection is unique. This function can be used to validate if a specific field, such as email or username, is already in use by another user.
Parameters:
Name Type Attributes Default Description
field string The field to check for uniqueness (e.g., "email", "username").
value string The value of the field to check for uniqueness.
userId string | null <optional>
 null Optional. The ID of the user to exclude from the uniqueness check. This is useful for updates, where the current user's field should not be considered a duplicate.
Source:
  • validation.js, line 23
Returns:
- Returns a promise that resolves to `true` if the field is unique, `false` otherwise.
Type
Promise.<boolean>
Examples
// Check if email is unique (for new users)
const isEmailUnique = await isUnique('email', 'user@example.com');
// Check if username is unique (for updating a user, excluding their own ID)
const isUsernameUnique = await isUnique('username', 'new_username', 'user_id_here');
