Movie API

This set of APIs will allow a user to get info about movies, directors, and writers, as well as create an account and add movies to their list of favorites.

API Overview

Welcome to our Basic Movie Information API. This simple yet effective API provides essential details about a range of movies, ideal for anyone interested in exploring film data.

Features

  • Move Details: Access information like titles, directors, cast, and genres.
  • Search Capability: Look up movie based on different criteria.
  • Genre Classification: Explore movies sorted by genre.

Target Users

  • Developers: Integrate movie data into your applications.
  • Movie Fans: Discover new films and learn more about your favorites.
  • Content Creators: Use movie data to enrich your content.

Quick Start

Explore our endpoints to get movie information quickly and efficiently. This API offers a straightforward way to access movie data, perfect for various applications and projects.

List of API Endpoints
Business Logic URL HTTP Method Request Body Format Response Body Format
Get a list of ALL movies to the user /movies GET None A JSON object holding data about all movies
Get information about a single movie by title /movies/{movieTitle} GET None

A JSON object holding data about s ingle film, containing a title, director, writer, cast, description, image URL, true or false value on whether it is currently featured, and genre.

Example:


{
  "title": "Vertigo",
  "director": ["Alfred Hitchcock"],
  "writer": ["Alec Coppel", "Samuel A. Taylor"],
  "cast": [
    "James Stewart",
    "Kim Novak",
    "Barbara Bel Geddes",
    "Tom Helmore",
    "Henry Jones"
  ],
  "description": "description of Vertigo",
  "imageUrl": "",
  "featured": true,
  "genre": ["suspense"]
}
Get a movie's director /movies/{title}/director GET None A JSON object containing one key, director, with a value that is an array of the director(s) of that movie. Example: 

{
  "director": ["Alfred Hitchcock"]
}
Get a movie's writer /movies/{title}/writer GET None A JSON object containing one key, writer, with a value that is an array of the writer(s) for that movie. Example: 

{
  "writer": ["Alfred Hitchcock"]
}
Get a movie's cast /movies/{title}/cast GET None A JSON object containing one key, cast, with a value that is an array of cast members of that movie. 

{
  "cast": [
  "James Stewart",
  "Kim Novak",
  "Barbara Bel Geddes",
  "Tom Helmore",
  "Henry Jones",
  ]
}
Get a movie's description /movies/{title}/description GET None A JSON object containing one key, description, with a value that is a string that is the description of that movie. 

{
  "Vertigo": "This is a description of Vertigo"
}
Get a movie's image url /movies/{title}/image GET None A JSON object containing one key, imageUrl, with a value that is a string of that url. Example: 

{
  "imageUrl": "https://en.wikipedia.org/wiki/Alfred_Hitchcock#/media/File:Hitchcock,_Alfred_02.jpg"
}
Get a movie's genre /movies/{title}/genre GET None A JSON object containing one key, genre, and a value that is an array containing strings of the movie's genre(s). 

{
  "genre": ["suspense", "thriller"]
}
Get information about a director /directors/{directorName} GET None

A JSON object holding data about a director including birth year, death year, a short bio, an array of films they directed, an image url, and a boolean of whether they are featured or not.

Example:


{
  "name": "Jean Vigo",
  "birthYear": 1905,
  "deathYear": 1934,
  "filmography": [
    "L'Atlante"
  ],
  "bio": "Vigo was born to Emily Clero and the militant anarchist Miguel Almereyda. Much of Vigo's early life was spent on the run with his parents. His father was imprisoned and probably murdered in Fresnes Prison on 13 August 1917, although the death was officially a suicide. Some speculated that Almereyda's death was hushed up on orders of the Radical politicians Louis Malvy and Joseph Caillaux, who were later punished for wartime treason.[1] The young Vigo was subsequently sent to boarding school under an assumed name, Jean Sales, to conceal his identity. Vigo was married and had a daughter, Luce Vigo, a film critic, in 1931. He died in 1934 of complications from tuberculosis, which he had contracted eight years earlier.",
  "imageUrl": "",
  "featured": true,
}
Get a description of a movie genre /genres/{genre} GET None A JSON object with a key, the name of the genre, and a value that is a string containing the description of the genre provided in the endpoint. 

  {
    "thriller": "A film or literary genre that uses various techniques to provoke a sense of suspense and excitement in the audience."
  }
Register a new user /users/ POST

A JSON object with a name property and a favoriteMovies property.

Example:


{
  "name": "Jane Smith",
  "favoriteMovies": []
}

A JSON object with an id, a name property and a favoriteMovies property.

Example:


{
  "id": 12345,
  "name": "Jane Smith",
  "favoriteMovies": []
}
Update user name /users/{id} PUT

A JSON object with an id, a name property and a favoriteMovies property.

Example:


{
  "id": 12345,
  "name": "Mike Tyson"
}

A JSON object with new user name

Example: 

{
  "id": 12345,
  "name": "Jane Smith",
  "favoriteMovies": []
}
Add a movie to a users favorite movie array /users/{id}/{movieTitle} PUT None String response confirming movie has been added.
Remove a movie to a users favorite movie array /users/{id}/{movieTitle} DELETE None String response confirming movie has been deleted.
Delete a user /users/{id}/ DELETE None String response confirming the user with the given ID has been deleted.
